# 風一族線上週報系統 (Royal Chosen Race Weekly)

這是一個基於 GitHub Pages 構建的靜態網站，專門用於存放與展示「風一族職場教會」的每週訊息、主日摘要及相關活動資訊。

## 🌐 網站資訊
* **網站網址：** [https://royal-chosen-race.github.io/weekly/](https://royal-chosen-race.github.io/weekly/)
* **核心宗旨：** 透過數位化平台分享每週主日訊息，使成員能隨時回顧屬靈教導與團體動態。

## ✨ 主要功能
* **週報存檔：** 按日期排列的每週訊息摘要。
* **主日信息：** 包含當週的主題、講員資訊及核心經文。
* **跨平台瀏覽：** 採用簡潔的網頁設計，支援手機與電腦端快速讀取。
* **靜態託管：** 利用 GitHub Pages 提供穩定且高速的訪問體驗。

## 📂 內容架構
網站內容通常包含以下模組：
1.  **Index/目錄頁：** 列出所有年份與月份的週報連結。
2.  **信息摘要：** 重點整理當週講道的精華內容。
3.  **報告事項：** 團體內部的近期活動預告或代禱事項。

## 🛠️ 技術實作
* **託管平台：** GitHub Pages
* **內容格式：** HTML / Markdown / Images
* **維護者：** Royal Chosen Race 媒體小組

## 🤝 聯繫與支持
如需獲取更多資訊或有內容勘誤需求，請訪問官方相關渠道或聯繫管理人員。
